# Financial Predictive Analysis Platform

Cette plateforme utilise le machine learning pour prédire les tendances des marchés financiers, en se concentrant spécifiquement sur les crypto-monnaies telles que le Bitcoin. Pour cela, le système recueille et analyse des données historiques provenant de différentes sources, telles que les prix passés, le volume de transactions, la volatilité, et d'autres indicateurs pertinents.

En utilisant ces données, le système entraîne des modèles de machine learning qui sont capables de reconnaître des schémas et des tendances dans les mouvements de prix. Ces modèles peuvent être basés sur des techniques telles que les réseaux de neurones artificiels, les arbres de décision, ou les méthodes ensemblistes comme XGBoost.

Pour améliorer la précision de la prédiction, la plateforme intègre également des indicateurs techniques comme l'Exponential Moving Average (EMA), le Relative Strength Index (RSI), le Moving Average Convergence Divergence (MACD), et d'autres. Ces indicateurs fournissent des informations supplémentaires sur la dynamique du marché et aident à affiner les prédictions du modèle.

L'objectif principal est de fournir aux utilisateurs des informations fiables et exploitables sur les tendances futures des prix des crypto-monnaies, ce qui peut les aider dans leurs décisions d'investissement et de trading.

# Financial Predictive Analysis Platform

This platform utilizes machine learning to predict financial market trends, focusing specifically on cryptocurrencies like Bitcoin. To do so, the system collects and analyzes historical data from various sources, such as past prices, transaction volume, volatility, and other relevant indicators.

Using this data, the system trains machine learning models capable of recognizing patterns and trends in price movements. These models can be based on techniques such as artificial neural networks (ANN), decision trees, or ensemble methods like XGBoost.

To improve prediction accuracy, the platform also integrates technical indicators such as Exponential Moving Average (EMA), Relative Strength Index (RSI), Moving Average Convergence Divergence (MACD), and others. These indicators provide additional insights into market dynamics and help refine the model's predictions.

The main goal is to provide users with reliable and actionable information about future price trends of cryptocurrencies, which can assist them in their investment and trading decisions.
